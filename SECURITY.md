## 报告漏洞

如果你发现任何安全漏洞，请**不要**在公开的 Issues 中报告。

请通过以下方式私下联系：

- 在 GitHub 上创建 [Issue](https://github.com/yosh-wang/auto_bluray-ISO-/issues) 并标注 `security`（会私下处理）
- 或直接联系仓库维护者

## 安全说明

本脚本仅做以下操作：

1. 读取 mpv 当前播放的文件路径
2. 检测是否为 `.iso` 文件
3. 替换播放协议为 `bd://`

脚本**不会**：
- 收集任何用户数据
- 访问网络
- 修改系统文件
- 执行任意代码

## 依赖风险

本脚本无任何外部依赖，仅使用 mpv 内置 Lua API。
